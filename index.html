ICD-10 工具總入口

ICD-10-CM 診斷索引、腫瘤表、藥物中毒表、外因表、PCS 一站式入口

這個 INDEX 頁面只是總入口，點選下方卡片會開啟對應的工具頁。 每個工具頁會各自從對應的 JSON 檔載入資料，例如： icd10_idx.json、icd10_desc.json、neoplasms.json、drug_idx.json、external_idx.json 等。

建議所有 .HTML 及 .json 檔案都放在同一個資料夾，以方便 GitHub Pages / Chrome 外掛使用。

ICD-10-CM 診斷索引

Diagnosis Index

依代碼或關鍵字查詢 ICD-10-CM 診斷。可搭配說明檔顯示完整敘述、Excludes、Includes 等資訊。

資料來源：icd10_idx.json（索引）＋ icd10_desc.json（詳細說明）
開啟診斷索引

腫瘤表 Neoplasms

Neoplasms Table

依部位與良惡性（malignant, benign, in situ, uncertain, unspecified）查詢對應腫瘤代碼。

資料來源：neoplasms.json
開啟腫瘤表

藥物與中毒表

Drug / Poisoning Index

依藥物或物質查詢不良反應、中毒、自殺、攻擊、未明等對應代碼。

資料來源：drug_idx.json
開啟藥物表

外因表 External Causes

External Causes Index

依意外事件、環境因素等關鍵字查詢對應外因代碼（跌倒、交通事故、暴力、自殺等）。

資料來源：external_idx.json
開啟外因表

ICD-10-PCS 處置／手術

Procedures (PCS)

依Section、Body System、Root Operation、Approach 等要素，查詢對應 ICD-10-PCS 處置代碼。

資料來源：PCS 專用 JSON（例如 pcs_idx.json 或其他命名，之後可自行補上）
